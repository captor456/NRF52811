# EPD-nRF5 项目源码分析文档

## 项目概述

**EPD-nRF5** 是一个基于 Nordic nRF5系列芯片的电子墨水屏(EPD)驱动固件项目。

### 主要特性
- **BLE蓝牙通信**：支持蓝牙低功耗无线连接
- **图像传输**：通过蓝牙传输图像到墨水屏
- **日历模式**：支持农历、节气、节假日调休显示
- **OTA升级**：支持蓝牙无线固件升级
- **低功耗设计**：支持深度睡眠和GPIO唤醒

### 支持的硬件

#### MCU芯片
- nRF51822 (Cortex-M0, 16K RAM, 128K/256K Flash)
- nRF51802 (Cortex-M0, 16K RAM, 256K Flash)
- nRF52810 (Cortex-M4, 24K RAM, 192K Flash)
- nRF52811 (Cortex-M4, 24K RAM, 192K Flash)

#### 电子墨水屏驱动IC
- **UC81xx系列**：UC8159, UC8176, UC8179
- **SSD16xx系列**：SSD1619, SSD1677
- **JD79xxx系列**：JD79668, JD79665

#### 颜色支持
- 黑白双色 (BW)
- 黑白红三色 (BWR)
- 黑白红黄四色 (BWRY)

## 核心模块分析

### 1. 主程序模块 (main.c)

#### 初始化流程
```
main()
├── log_init()              # 初始化日志系统
├── 保存复位原因
├── WDT初始化               # 看门狗
├── timers_init()           # 定时器
├── power_management_init() # 电源管理
├── ble_stack_init()        # BLE协议栈
├── scheduler_init()        # 事件调度器
├── gap_params_init()       # GAP参数
├── gatt_init()             # GATT (nRF52)
├── services_init()         # 服务初始化
├── advertising_init()      # 广播初始化
├── conn_params_init()      # 连接参数
├── application_timers_start() # 启动定时器
├── advertising_start()     # 开始广播
└── 主循环
    ├── app_sched_execute() # 执行调度事件
    └── idle_state_handle() # 空闲处理(喂狗+睡眠)
```

#### 关键功能
- **时钟管理**：每秒更新时间戳，触发显示刷新
- **看门狗**：30秒喂狗一次，防止死机
- **睡眠管理**：广播超时后进入深度睡眠
- **唤醒机制**：支持GPIO中断唤醒

### 2. EPD驱动模块 (EPD_driver)

#### 驱动架构
```
epd_driver_t (驱动函数表)
├── init()         # 初始化
├── clear()        # 清屏
├── write_image()  # 写图像
├── write_ram()    # 写RAM
├── refresh()      # 刷新显示
├── sleep()        # 进入睡眠
├── read_temp()    # 读取温度
└── read_busy()    # 读取忙状态
```

#### GPIO引脚定义
| 引脚 | 功能 | 说明 |
|------|------|------|
| MOSI | SPI数据输出 | 主出从入 |
| SCLK | SPI时钟 | 同步时钟 |
| CS | 片选 | 低电平有效 |
| DC | 数据/命令选择 | 0=命令, 1=数据 |
| RST | 复位 | 低电平复位 |
| BUSY | 忙信号 | 高电平表示忙 |
| BS | 总线选择 | 部分屏幕使用 |
| EN | 电源使能 | 控制屏幕电源 |
| LED | 状态LED | 指示运行状态 |

#### SPI通信协议
```c
// 写命令
void EPD_WriteCmd(uint8_t cmd) {
    digitalWrite(EPD_DC_PIN, LOW);  // DC = 0 (命令)
    EPD_SPI_Write(&cmd, 1);
}

// 写数据
void EPD_WriteData(uint8_t* value, uint8_t len) {
    digitalWrite(EPD_DC_PIN, HIGH); // DC = 1 (数据)
    EPD_SPI_Write(value, len);
}
```

### 3. BLE服务模块 (EPD_service)

#### GATT服务结构
```
EPD Service (UUID: 62753622-...-675AEC)
├── EPD Characteristic (UUID: 0x0002)
│   ├── 属性: 可读、可写、可通知
│   └── 功能: 接收命令、发送响应
└── Version Characteristic (UUID: 0x0003)
    ├── 属性: 可读
    └── 功能: 读取固件版本
```

#### 命令协议

| 命令 | 代码 | 功能 |
|------|------|------|
| SET_PINS | 0x00 | 设置引脚映射 |
| INIT | 0x01 | 初始化屏幕 |
| CLEAR | 0x02 | 清屏 |
| SEND_COMMAND | 0x03 | 发送原始命令 |
| SEND_DATA | 0x04 | 发送原始数据 |
| REFRESH | 0x05 | 刷新显示 |
| SLEEP | 0x06 | 屏幕睡眠 |
| SET_TIME | 0x20 | 设置时间戳 |
| SET_WEEK_START | 0x21 | 设置星期起始 |
| WRITE_IMAGE | 0x30 | 传输图像 |
| SET_CONFIG | 0x90 | 设置配置 |
| SYS_RESET | 0x91 | 系统复位 |
| SYS_SLEEP | 0x92 | 系统睡眠 |
| CFG_ERASE | 0x99 | 擦除配置 |

#### 图像传输协议
```
[命令: 0x30] [配置字节] [图像数据...]

配置字节格式:
Bit 7-4: 数据类型 (0=黑白, 1=红色/黄色)
Bit 3-0: 包序号 (用于分包传输)
```

## 关键代码分析

### 1. 看门狗管理
```c
// 全局变量
static nrf_drv_wdt_channel_id m_wdt_channel_id;
static uint32_t m_wdt_last_feed_time = 0;

// 喂狗函数
void app_feed_wdt(void) {
    // 每30秒喂一次
    if (m_timestamp - m_wdt_last_feed_time >= 30) {
        nrf_drv_wdt_channel_feed(m_wdt_channel_id);
        m_wdt_last_feed_time = m_timestamp;
    }
}

// 在空闲循环中调用
static void idle_state_handle(void) {
    app_feed_wdt();  // 喂狗
    if (NRF_LOG_PROCESS() == false) 
        nrf_pwr_mgmt_run();  // 进入低功耗
}
```

### 2. 睡眠管理
```c
// 进入深度睡眠
void sleep_mode_enter(void) {
    NRF_LOG_DEBUG("Entering deep sleep mode\n");
    NRF_LOG_FINAL_FLUSH();
    nrf_delay_ms(100);
    
    // 准备EPD睡眠
    ble_epd_sleep_prepare(&m_epd);
    
    // 关闭系统
    nrf_pwr_mgmt_shutdown(NRF_PWR_MGMT_SHUTDOWN_GOTO_SYSOFF);
}

// 唤醒引脚设置
static void setup_wakeup_pin(nrf_drv_gpiote_pin_t pin) {
    nrf_drv_gpiote_init();
    // 配置为上升沿触发
    nrf_drv_gpiote_in_config_t config = GPIOTE_CONFIG_IN_SENSE_LOTOHI(false);
    nrf_drv_gpiote_in_init(pin, &config, gpiote_evt_handler);
    nrf_drv_gpiote_in_event_enable(pin, true);
}
```

### 3. 电压检测
```c
uint16_t EPD_ReadVoltage(void) {
#if defined(S112)
    // nRF52使用SAADC
    NRF_SAADC->RESOLUTION = SAADC_RESOLUTION_VAL_10bit;
    NRF_SAADC->ENABLE = (SAADC_ENABLE_ENABLE_Enabled << SAADC_ENABLE_ENABLE_Pos);
    NRF_SAADC->CH[0].PSELP = SAADC_CH_PSELP_PSELP_VDD;  // 测量VDD
    // ... 采样过程
    return (value * 3600) / (1 << 10);  // 转换为mV
#else
    // nRF51使用ADC
    NRF_ADC->CONFIG = (ADC_CONFIG_RES_10bit << ADC_CONFIG_RES_Pos) |
                      (ADC_CONFIG_INPSEL_SupplyOneThirdPrescaling << ADC_CONFIG_INPSEL_Pos);
    // ... 采样过程
    return (value * 3600) / (1 << 10);
#endif
}
```
